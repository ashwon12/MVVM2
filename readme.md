<br><br>

## 📌 Technology Stack
- MVVM 아키텍처 패턴
- Android AAC (LiveData, ViewModel, Repository)
- DataBinding
- 네트워크 : Retrofit2 , OkHttp3 , Gson
- 이미지 처리 : Glide
- 의존성 주입 : Hilt
